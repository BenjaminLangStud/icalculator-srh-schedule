# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.1.x   | :white_check_mark: |
| 1.0.0   | :x:                |

## Reporting a Vulnerability

1. Think about it: is this really neccessary to patch
2. Cry about it
3. Creat an issue (yes, the public one)
4. Sit back and wait, maybe we'll try to fix it at some point. No hurries tho
